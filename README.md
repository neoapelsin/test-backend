## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

```

## Env

```bash

DATABASE_URL={url}

```

## Request example

```bash
http://localhost:3000/user/balance/1

data: {
  amount: -2
}
```

## Test

```bash
# unit tests
$ npm run test

```

## Installation

```bash
$ npm install
```

## Running the app

```bash
# build
$ npm run build

# prod
$ npm run start:prod

# watch mode
$ npm run start:dev

```

## Env

```bash

DATABASE_URL={url}

```

## Request example

```bash
POST: http://localhost:3000/api/user/balance/1

data: {
  "amount": -2
}
```
